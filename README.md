## Laboratory Session 5: Statistical Analysis

This lab focuses on fundamental statistical analysis techniques using Python. Students will learn to compute and visualize key statistical measures using NumPy, pandas, and Matplotlib.

### Learning Objectives

- Generate and visualize random datasets using Line Charts
- Calculate statistical measures: Mean, Median, Range, and Standard Deviation
- Work with pandas DataFrames and compare with NumPy arrays
- Use pandas `describe()` method for descriptive statistics
- Compute quartiles and Interquartile Range (IQR)

### Libraries Used

- **NumPy** - Numerical computations and statistical functions
- **pandas** - Data manipulation and analysis
- **Matplotlib** - Data visualization

### Tasks Overview

| Task | Description                                                          |
| ---- | -------------------------------------------------------------------- |
| 1    | Create random dataset and visualize with Line Chart                  |
| 2    | Calculate Mean, Median, Range, Standard Deviation using NumPy        |
| 3    | Convert array to pandas DataFrame and compare outputs                |
| 4    | Use `describe()` for statistical summary (mean, quartiles, median) |
| 5    | Calculate Interquartile Range (IQR)                                  |
